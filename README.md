# Shein Telegram Tracker
This space runs a high-speed Telegram bot to track Shein India category drops.

## Setup
1. Set the following **Secrets** in your Space settings:
   - `TELEGRAM_BOT_TOKEN`
   - `ADMIN_ID`
   - `SHEIN_COOKIES`
   - `WEBHOOK_URL` (optional)
